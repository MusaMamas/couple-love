# Couple Love
App where couple would test their comparisons in interest and other functions, which can enhance couple's relationship.

## Käytetyt teknologiat

React Native + Expo
Projektin käyttöliittymä on rakennettu React Nativen avulla, ja Expo toimii kehitysympäristönä ja sovelluksen ajonaikaisena alustana.

TypeScript
Sovelluslogiikka ja komponentit on kirjoitettu TypeScriptillä, joka parantaa koodin turvallisuutta ja kehityskokemusta.

Firebase
Projekti sisältää firebaseConfig.ts-tiedoston, joten käytössä on Firebase (yleensä Authentication, Firestore ja/tai Storage).
